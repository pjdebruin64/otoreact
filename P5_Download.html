OtoReact Download, license, installation

Downloads

OtoReact is basically just a single 38KB file which you can download here: OtoReact.js
Here is the alternative version for older browsers, targeting EcmaScript 2015: OtoReact_ES2015.js
if you are using TypeScript, then you may want these type declarations: OtoReact.d.ts
Here is the OtoLib library: OtoLib.html

If your application uses Routing on pathnames, then you need to configure your server to serve the same HTML file on different pathnames.

On Microsoft IIS, it suffices to place this file in your application root: web.config
On Apache HTTP Server, it suffices to place this file in your application root, and rename it to ".htaccess": htaccess.txt

Requirements

Server

Otoreact does not have server requirements: any web (http/https) server serving static files will do, like:
  • Microsoft IIS can easily be enabled on most Windows systems
  • Apache HTTP Server is available for both Windows and Linux
  • Nginx also, and takes the least system resources

If you need a small local server just for development, then some options are:

  • The Visual Studio Code editor has plenty of web server extensions that can serve your files during development, and automatically refresh your page when you save changes.
    The most popular one is Live Server
  • The node.js package serve allows you to temporarily run a small web server
  • In former days, web browsers could open local websites straight from disk without needing an http server.

    Nowadays, web security rules prohibit this. One can however temporarily disable these rules.

    For Google Chrome this can be done by running:

    "%ProgramFiles(x86)%\Google\Chrome\Application\chrome.exe" --disable-web-security --user-data-dir=%Tmp%
    Then drag your html file to the new window, or add its path to the command above.

Clients

OtoReact is written in JavaScript and requires a reasonably modern web browser to run. It does not run in Internet Explorer.
If you can read this text, then OtoReact is running in your current browser.

The standard version is compiled for ECMAScript 2019 and runs in any modern browser version released in 2019 or later.

There is an alternative version compiled for ECMAScript 2015 (ES5) that should run in older browsers supporting ES5.
Note that OtoReact is not being tested in older browsers, and performance may be bad.
