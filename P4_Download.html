OtoReact Download, license, installation

Requirements

Server

Otoreact does not have any server requirements: any web server serving static files will do.

If you need a small local server just for development, then the npm 'serve' package will serve you well.

Clients

OtoReact is heavy on JavaScript and requires a reasonably modern web browser to run. It does not run in Internet Explorer.
If you can read this text, then OtoReact is running in your current browser.

The standard version is compiled for ECMAScript 2019 and runs in any modern browser version released in 2019 or later.

There is an alternative version compiled for ECMAScript 2015 (ES5) that should run in older browsers supporting ES5.
Note that OtoReact is not being tested in older browsers, and performance may be bad.

Downloads

OtoReact is basically just a single file which you can download here: OtoReact.js

if you are using TypeScript, then you may want these type declarations: OtoReact.d.ts

Here is the alternative version for older browsers: OtoReact_ES5.js

Here is the OtoLib library: OtoLib.html

Routing configuration

If your application uses Routing on pathnames, then you need to configure your server to serve the same HTML file on different pathnames.
On Microsoft IIS, it suffices to place this file in your application root: web.config
On Apache HTTP Server, it suffices to place this file in your application root: htaccess.txt and rename it to ".htaccess".
